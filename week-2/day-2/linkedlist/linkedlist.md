### Questions

why linkedlist over a python list?

- inserting or deleting element at the beginning of a linkedlist is O(1) time. For python lists, shifting elements takes O(n) time

What is the main weakness of a linked list?

- No random access. Accessing i-th element requires tracking from the head pointer which takes O(n) time.
  Also bad for cache efficiency because memory is allocated non contiguously
